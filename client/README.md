# GazeType — client

The React + Vite + TypeScript frontend for **GazeType**.

For full documentation — architecture, local setup, environment variables, and
deployment — see the [project README](../README.md) and [DEPLOY.md](../DEPLOY.md)
at the repository root.

## Quick start (from the repo root)

```bash
npm run dev          # runs client + server together
```

Or run just the client:

```bash
npm run dev --prefix client   # http://localhost:5173
```
